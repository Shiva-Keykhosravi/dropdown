Dropdown Button (HTML & CSS)

A minimal and clean dropdown button component built using HTML, CSS, and Flexbox.
The dropdown opens on hover, includes a smooth animation, and uses Google Fonts for a modern look.

Features :

Pure HTML + CSS (no JavaScript)
Flexbox layout
Hover-based dropdown animation
Smooth opacity & translate transitions
SVG icon rotation on open
Centered and responsive layout
